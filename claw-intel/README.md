# 🕵️ claw-intel · 比价情报员

> 大品牌商品多平台比价，30秒出一份可以直接发微信群的报告。

---

## 能做什么

- 自动抓取京东、天猫、拼多多、抖音、小红书多平台价格
- 强制 3 源交叉验证，带时间戳防假价
- 输出带表格 + emoji 的漂亮报告，核心结论 + 价格对比 + 买/等建议
- 直接截图发微信群

## 适合场景

✅ **效果最好：** 苹果、戴森、索尼、耐克等大品牌标品，型号明确的 3C、家电、运动装备

⚠️ **效果一般：** 需要登录才能看的会员价、直播间限时价（直播不在线时）

❌ **不适合：** 淘宝服装等非标品、小众品牌

---

## 安装方法

```bash
mkdir -p ~/.openclaw/workspace/skills/claw-intel
# 下载 SKILL.md 放入上述文件夹
openclaw gateway restart
```

---

## 使用方法

装好后直接发：

```
情报局，比价 iPhone 17 Pro Max 1TB 最新价
帮我查一下戴森吹风机哪个平台最便宜
Switch 2 现在多少钱，值得买吗
```

不需要说"比价"，只要涉及购买决策，龙虾会自动触发。
